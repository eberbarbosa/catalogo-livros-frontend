🎯 Sobre o Projeto

<<<<<<< HEAD
O Catálogo de Livros é o frontend de um sistema completo para gestão e descoberta de livros. A proposta é simples: entregar uma interface limpa, rápida e intuitiva, pronta para se conectar a qualquer API backend e exibir dados reais de títulos, autores, capas e sinopses.O que torna esse projeto diferente:
🧠 Arquitetura pensada para crescer — modular e escalável desde o início
⚡ Build ultrarrápido com Vite, sem abrir mão de boas práticas
🎨 UI consistente com componentes reutilizáveis e estilização organizada
🔌 Pronto para integrar com qualquer API REST

  71+ commits de evolução contínua e contagem…
=======
O Catálogo de Livros nasce da necessidade de ter uma plataforma centralizada e intuitiva para gerenciar acervos literários. Seja para uso pessoal, bibliotecas ou livrarias, a aplicação oferece uma experiência fluida de ponta a ponta.Por que este projeto?
📌 Interface pensada para o usuário final, sem complexidade desnecessária
🔒 Código limpo, organizado e escalável
⚙️ Arquitetura modular com componentes reutilizáveis
🌐 Pronto para integrar com qualquer API REST backend

  💡 Projeto em desenvolvimento ativo com mais de 71 commits de evolução contínua.
